# Silver Eagle
## nazewnictwo plików camelCase ( jakasNazwaZmiennej )

## 📂 Struktura projektu
```
Assets/  
├── Art/ # Wszystkie assety wizualne  
│ ├── Characters/ # Modele postaci + tekstury  
│ ├── Environments/ # Sceny, teren, budynki, rośliny  
│ ├── Props/ # Dekoracje, obiekty interaktywne  
│ ├── Animations/ # Animacje + kontrolery animatora  
│ ├── Materials/ # Materiały do modeli  
│ ├── Textures/ # Tekstury (PNG, JPG)  
│ └── FX/ # Efekty cząsteczkowe i VFX  
│  
├── Audio/ # Wszystkie pliki dźwiękowe  
│ ├── Music/ # Ścieżka dźwiękowa  
│ ├── SFX/ # Efekty dźwiękowe  
│ └── Voices/ # Dialogi / narracja  
│  
├── Prefabs/ # Gotowe prefabrykaty do użycia w scenach  
├── Scenes/ # Wszystkie sceny Unity  
├── Scripts/ # Skrypty C#  
├── UI/ # Interfejs użytkownika  
│ ├── Canvases/  
│ ├── Sprites/  
│ └── Fonts/  
│  
├── Plugins/ # Zewnętrzne biblioteki i pakiety  
└── Tests/ # Testy jednostkowe i integracyjne  
```
